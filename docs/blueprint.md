# **App Name**: LevelUp List

## Core Features:

- Task Management: Create and manage tasks with assigned XP values.
- Progress Tracking: Visually track XP accumulation and level progression with a dynamic level bar.
- Smart XP Suggestion: An AI tool that automatically suggests relevant XP values for new tasks, to ensure tasks have proportionate value.
- Task Completion: Option to mark tasks as complete, updating the XP bar in real time.
- Profile display: A profile page which stores all completed tasks and current level and current XP of the User
- Motivation: Motivational display which acknowledges the user after reaching certain level or experience thresholds
- Task Sections: There are two sections in the tasks: one is daily tasks which resets on every day, and the other does not have a deadline

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to represent progress and achievement.
- Background color: Light gray (#F0F2F5) for a clean, uncluttered workspace.
- Accent color: Warm orange (#FF8C00) to highlight the XP bar and CTAs.
- Body and headline font: 'Inter' sans-serif for a clean and modern feel.
- Use clear, minimalist icons for task categories and completion status.
- Maintain a clean, list-based layout for easy task management.
- Incorporate smooth transitions and celebratory animations for level-ups and task completions.
